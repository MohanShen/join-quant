# 解剖报告：2f5bc859_四择时大小盘风格轮动

## 一句话结论
一个 3 择时 **大小盘 style-vote 切换器**（择时一 大小盘涨幅博弈 / 择时二 沪深300-vs-国证2000 比值 / 择时三 小市值一致性 → 投票选 白马大市值 / 高息低价小盘 / 国债ETF避险，叠加日频 MACD顶背离 dapan 风控 + 市场温度-conditional 白马财务过滤，月度调仓）；基线 **MARGINAL 过门槛**（Sharpe 2.67、obj 0.4564，家族横评垫底带）。核心发现：**它的 STYLE VOTE 是净拖累**——强制 always-(高息低价小盘) 把 objective 0.4564→0.6656（**+0.21**、年化 +22pt、Sharpe +0.85）而仅 **+1.2pt maxDD**；投票切向 白马/empty 用一丁点回撤削减换掉大量收益。定性上更接近 [[9f6f8188_大小盘多策略]] 的**严格稀释**而非 [[1c80dd9a_大小外择时小市值]] 的**真实但昂贵保护**。⚠ 零滑点高估（小盘 sleeve 高换手）+ regime-conditional（2022-23 偏好小盘）。

## regime 依赖
drag 判定是 **window-specific**：冻结窗（2022 熊 + 2023 震荡）恰**偏好小盘**，故投票切向 白马大市值 / 国债ETF避险都在付掉收益。在**小盘崩盘 regime**（如 2024-02 微盘踩踏，本窗未含）里，白马 sleeve + MACD顶背离风控 + empty 避险切换**可能真正兑现保护**、objective 也可能翻正。跨 regime 普适性 med（子窗分区间 un-run）。

## 机理与真实性
- ⚠ **零滑点高估**：小盘（高息低价小盘）sleeve 高换手，零滑点**高估** always-small 的收益增幅——方向稳健，量级虚高。
- **消融隔离干净**：ablation 中 **MACD顶背离 risk-off 保持 ON**，故本实验隔离的是 big-vs-small **STYLE 投票**，**不是** dapan 风控。风控腿与市场温度 白马 财务过滤的独立贡献未拆解。

**反哺 auto-research**：
1. **多择时 style-vote（强小盘 sleeve + 白马/empty 切换）在小盘主导 regime 里倾向对 annual−maxDD 净拖累**——跨 [[9f6f8188_大小盘多策略]]（严格稀释）/ [[1c80dd9a_大小外择时小市值]]（真实但昂贵保护）/ 本 target 递归复现。ideator 生成「加大盘/白马/避险 sleeve 做风格切换」类点子时，必须由一次 out-of-regime（小盘崩盘窗）保护测试证成，勿默认多样化。
2. **本策略的 marginal objective 是 style-vote artifact**——纯高息低价小盘 sleeve 独立即得 obj 0.67；诊断弱 style-switch 策略先查投票切换的收益代价，再怀疑组件损坏。区分「严格拖累」（去掉每轴皆改善，9f6f8188）与「真实但昂贵保护」（去掉 obj 升 AND 回撤明显翻倍，1c80dd9a）——本 target 属前者的弱化版（回撤只涨 1.2pt，保护太薄不值收益代价）。
