_创建于 2026-05-04_

## 策略概述

**核心理念**：专门捕捉ST股的"弱势转强势"拐点。即挑选那些之前弱势/跌势的ST股，在它们开始转强时买入。

**回测表现**：
- 回测周期：2020-01-01 → 2025-03-31（5年+）
- 初始资金：¥100,000
- 频率：分钟级
- 累计收益：最高约 22,500%（225倍）
- 克隆数：323人

## 选股流程（prepare 函数）

### 第一步：时间过滤
只在特定月份操作，避免年末和季末：
- 1月15日 ~ 1月31日
- 4月15日 ~ 4月31日
- 12月15日 ~ 12月31日

### 第二步：国九条过滤（GJT_filter_stocks）
排除财务不合格的股票：
- 净利润（归属于母公司）> 0
- 净利润 > 0
- 营业收入 > 1亿元
- ROE > 0
- ROA > 0

### 第三步：技术指标筛选（filter_stocks）
- 收盘价 > 前一日最低价（企稳信号）
- 收盘价 > 10日均线（多头排列）
- 今日成交量 > 前一日成交量（温和放量）
- 今日成交量 < 10倍前一日成交量（避免暴增）
- 股价 > 1元

### 第四步：低开过滤
前一日收盘 vs 今日开盘：
- 开盘价/收盘价 在 0.95 ~ 1.01 之间
- 过滤掉大幅低开和高开的股票

### 第五步：换手率排序
按换手率降序排列，取前5只

---

## 买入逻辑（buy 函数）

1. **等权重分配**：可用资金 / 买入数量
2. **排除停牌股**
3. **排除涨跌停股**（无法买入）

---

## 卖出逻辑（sell 函数）

满足以下任一条件即卖出：

| 条件 | 说明 |
|------|------|
| 未涨停 | 现价 ≠ 涨停价 |
| 亏损 > 3% | 成本价 vs 现价 |
| 盈利 > 0% | 超过成本价 |
| 昨日涨停 | 涨停后可能回调 |

**注意**：使用 `order_target_value(s, 0)` 清仓

## 策略特点

### 优点
1. **择时明确**：只在特定月份操作，避免高风险期
2. **风险控制**：亏损3%即止损，不持有亏损股
3. **止盈灵活**：盈利即出，不贪
4. **仓位管理**：最多5只，等权重分散

### 风险
1. **ST股波动大**：可能连续跌停无法卖出
2. **流动性风险**：小市值ST股可能无法按目标价买入
3. **政策风险**：ST制度可能变化
4. **高换手**：每年操作月份集中，可能错过趋势
